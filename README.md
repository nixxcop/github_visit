# GitHub Visit Automation (Colab Ready)

This repository provides a Google Colab–ready notebook using Playwright and asyncio to simulate controlled web visits. It supports batching, logging, and headless execution with minimal setup. Ideal for testing, automation experiments, and learning async browser workflows.

## ▶ Open in Google Colab
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/nixxcop/github_visit)

## Requirements
Runs entirely on Google Colab. No local installation required.
